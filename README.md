## 프로젝트 소개
- 본 프로젝트는 2024.08.19 ~ 2024.08.20 진행한 해커톤 프로젝트 입니다.
- LangChain 기술을 활용하여 학교 종합강의시간표 PDF파일을 업로드 하면 LLM 모델이 이를 학습하여 사용자에게 맞는, 전공 커리큘럼을 제공하는 서비스

## 담당 업무
- 학습에 필요한 시간표 데이터를 JSON 형식으로 구조화 하는 작업을 수행하였습니다.
- 서비스 발표 자료를 제작하였습니다.

## 해커톤 회고
- **인생에서 처음으로 나가보는 해커톤**<br>
당시에는 개발자로서의 일을 희망해왔지만, 정확히 무슨일을 해야할지 깊이있게 고민해보지는 않았었습니다.<br><br>
그렇다 보니 백엔드로서의 기술적인 지식이 전무한 상황이라 본 해커톤에서 할 수 있는 일이 제한적이었습니다.

- **깨달음**<br>
저를 포함 총 4명과 함께 기획부터 프로젝트를 진행해나갔습니다.
학교 선배님들과의 첫 협업이었는데, 2학년이다보니 아직 웹과 관련된 기술들을 처음 배우기 시작하는 단계였고 멋모르고 흘러간 해커톤이었기 때문에 많은 부분들이 미숙하였습니다.<br><br>
그래도 웹프로그래밍 수업을 들었기 때문에 뭐라도 할 수 있는 부분이 있겠지 라는 생각이였으나 실상은 AI 훈련에 필요한 데이터를 JSON 형식으로 구조화하는 것 이외에는 할 수 있는 부분이 없었고, 실질적으로 개발에 참여하기 보다는 발표 자료나 서포트 하는 역할을 좀 더 많이 했었습니다.<br><br>
지금 돌아보면 2학년 후배와 함께 곧 졸업하는 선배님들께서 정말 많이 고생하셨을 것 같다 라는 생각과 조금 섣부르게 해커톤에 지원 한 것이 아닌가 라는 생각도 드는 것 같습니다.<br><br>
해커톤을 마무리 하고 나서 든 생각은, "정말 많은 공부가 필요하겠구나" 그리고 "내가 못하면 남들이 고생하는구나" 라는 사실을 뼈져리게 상기하는 시간이 되었습니다.<br><br>

- **앞으로의 다짐**<br>
공부를 정말 열심히 해서 앞으로 나가는 해커톤에서는 한명의 개발자로서 기여를 해보자! 라고 다짐하였습니다.
